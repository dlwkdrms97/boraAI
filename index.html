🚀 시작하기

1. 하단 [수업 시작] 버튼을 클릭하고 마이크 권한을 허용하세요.
2. 중다기초선 설계 연구를 위해 지연시간 목표(예: 10초)를 하단에 입력하세요.
3. 국어 수업 예시: "책을 펴세요"라고 말하면 큰 이모지가 나타나며 측정이 시작됩니다.

📊 연구 데이터 기록 (지연시간 측정)

T0 (시작): "앉으세요", "펴세요", "보세요" 등 지시어가 감지되면 녹색 바가 줄어듭니다.
T1 (종료): 학생이 행동을 수행했을 때 "잘했어요", "최고", "맞아요" 등의 칭찬어를 말씀하세요.
결과: 칭찬어를 하는 즉시 바가 파란색으로 변하며 0.1초 단위의 지연시간이 자동 저장됩니다.

🔑 Gemini AI API 키 발급 방법

  1. Google AI Studio에 접속하여 로그인합니다.
  2. 좌측 상단 [Get API key] 메뉴를 클릭합니다.
  3. [Create API key in new project] 버튼을 눌러 키를 생성합니다.
  4. 생성된 키를 복사하여 BORA의 설정(⚙️) > AI 탭에 붙여넣고 저장하세요.
