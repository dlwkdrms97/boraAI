💡 보라AI 완벽 활용 가이드

BORA는 시각적 촉구와 반응 시간 측정을 결합한 ABA 연구 도구입니다.

1. 스마트한 기록 및 테마 전환

• 단축키: 지시 카드가 떴을 때 Enter 또는 Space 키로 즉시 완료 기록이 가능합니다.
• 테마: 우측 하단 🌙/☀️ 버튼으로 교실 환경에 맞는 다크/라이트 모드를 선택하세요.

2. 데이터 신뢰성 및 요약 확인

• 정밀 측정: 지시 직후 0.8초간 음성 인식을 방어하여 중복 트리거를 방지하고 정확한 지연 시간만 기록합니다.
• 실시간 요약: 좌측 [오늘 지시 요약]에서 지시어별 수행 횟수를 실시간으로 확인하세요.

3. ABA 촉구 용암법(Fading) 적용

설정에서 촉구 단계를 조절하세요. 학생의 습득도에 따라 [텍스트 제거]나 [크기 축소]를 적용하여 시각적 단서를 점진적으로 제거할 수 있습니다.

4. 수업 종료 및 자동 저장

수업 종료 시 나타나는 팝업을 통해 [학생이름_날짜] 형태의 파일명으로 이번 세션 기록만 깔끔하게 별도 저장할 수 있습니다.

5. 마이크 권한 고정 (필수 설정)

10분마다 뜨는 권한 팝업을 없애려면:
• 주소창 왼쪽의 [자물쇠] 또는 [설정] 아이콘 클릭
• 마이크 항목을 '확인'에서 '허용'으로 변경
• 이제 브라우저를 껐다 켜도 팝업 없이 바로 기록됩니다.
