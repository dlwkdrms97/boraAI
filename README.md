# BORA (Behavior · Observation · Reaction · Achievement)
### AI 기반 특수교육 수업 지원 및 지연시간(Latency) 측정 시스템

BORA는 음성 인식 인공지능을 활용하여 발달장애 학생의 시각적 지원을 자동화하고, 행동 수정의 핵심 지표인 지연시간을 정밀하게 측정하는 보조 공학 도구입니다.

## 🚀 주요 기능
- **실시간 하이브리드 음성 분석**: 로컬 DB와 Google Gemini AI를 결합하여 교사의 발화를 시각 정보(이모지+텍스트)로 즉시 전환
- **지연시간(Latency) 자동 측정**: 시각 단서 제시 후 학생의 반응(교사의 긍정 강화 트리거)까지의 시간을 0.1초 단위로 기록
- **연구 데이터 관리**: 누적된 측정 데이터를 CSV 형식으로 내보내어 행동 분석 연구에 활용 가능
- **개별화 설정**: 학생별 이름, 지연 시간 목표, API 키 등을 브라우저에 안전하게 저장

## 🛠️ 연구대회 활용 안내
- 본 프로젝트는 **제65회 전국 특수교육 연구대회** 출품을 목적으로 개발되었습니다.
- 인공지능 음성 인식을 통한 시각적 피드백 제공이 지적장애 및 자폐성 장애 학생의 수업 참여도에 미치는 효과를 검증하는 데 최적화되어 있습니다.

이 프로젝트는 특수교육 현장의 혁신을 위해 공개되었습니다. 마이크 권한 유지를 위해 반드시 HTTPS 환경(GitHub Pages 등)에서 실행하는 것을 권장합니다.
